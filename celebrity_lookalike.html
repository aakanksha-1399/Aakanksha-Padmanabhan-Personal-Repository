You're My Celebrity

React AWS Rekognition AWS Lambda Amazon S3 AWS API Gateway AWS DynamoDB AWS Amplify AWS CloudWatch Postman Terraform

Project Overview

This project involved building a serverless web application that lets users upload selfies and find their celebrity lookalikes using AWS Rekognition. It combines an intuitive React front-end with a scalable backend powered by AWS Lambda, API Gateway, and DynamoDB for fast processing and data storage. The app also features automated deployment via Terraform and performance monitoring with CloudWatch to ensure smooth and reliable operation.

Features & Highlights

This project incorporates a variety of advanced features designed to enhance user experience and system performance. From intuitive interfaces to optimized backend processing, the highlights demonstrate a scalable, efficient, and user-focused application architecture.

  • User-friendly image upload interface built with React for seamless use on both desktop and mobile devices.
  • Celebrity lookalike matching powered by AWS Rekognition, comparing uploaded photos with a curated dataset of over 160 celebrities.
  • Real-time feedback mechanisms provide users with progress updates and processing status.
  • Optimized serverless backend using AWS Lambda for image preprocessing, improving speed by 40%.
  • Secure, scalable architecture utilizing AWS API Gateway, S3, and DynamoDB for efficient data management.
  • Persistent user match history stored in DynamoDB, allowing easy access to past results.
  • Comprehensive monitoring and logging with AWS CloudWatch for performance tracking and error detection.
  • Automated deployment pipeline built with Terraform and AWS Amplify for smooth updates and infrastructure management.

Architecture & Performance

The application follows a fully serverless architecture using AWS services like API Gateway, Lambda, S3, DynamoDB, and Rekognition. Terraform automates infrastructure provisioning, while AWS Amplify ensures seamless frontend deployment. CloudWatch monitors system performance, enabling real-time visibility and faster debugging.

Image processing was optimized by introducing Lambda-based preprocessing, reducing latency by 40%. Asynchronous request handling further improved response times, while confidence thresholds in Rekognition were fine-tuned for better celebrity match accuracy.

Deployment & CI/CD

The application is deployed using a smooth and fully automated CI/CD pipeline. Terraform provisions and manages infrastructure, while AWS Amplify handles build, deployment, and hosting. Results

The application successfully matches user-uploaded selfies with celebrity lookalikes using AWS Rekognition, delivering fast and accurate results through a serverless, cloud-native pipeline.
