Book Recommendation System

Python Flask Pandas Scikit-learn Machine Learning Cosine Similarity SQLite

Project Overview

This web-based book recommendation system leverages machine learning to suggest titles based on both popularity metrics and user preferences. Using cosine similarity and content-based filtering, it delivers personalized recommendations tailored to individual input. Built with Python, Flask, and scikit-learn, the application features a clean, interactive interface for seamless search and real-time suggestions.

Data Overview

The project leverages data-driven techniques to deliver insightful book recommendations by analyzing user behavior and preferences. Through effective data processing and machine learning algorithms, it demonstrates my ability to extract meaningful patterns and build user-centric models.

  • Identified the top 50 popular books using comprehensive analysis of user ratings and reviews.
  • Applied cosine similarity for personalized recommendations based on user-inputted book rankings.
  • Developed an interactive Flask-based front-end to visualize and deploy the recommendation system seamlessly.

Popularity-Based Book Recommendation

In this module, I developed a popularity-driven book recommendation system by analyzing user ratings and reviews. The system filters books with at least 250 ratings to ensure reliable popularity metrics, then ranks them by average rating to highlight the top 50 most favored titles. Data preprocessing involved merging multiple datasets and aggregating ratings to create meaningful features. This approach demonstrates my ability to engineer and clean large datasets, extract relevant insights, and implement business rules to deliver interpretable recommendations.

  • Merged and cleaned data from multiple sources including user ratings and book metadata.
  • Applied filtering thresholds to ensure statistical significance in recommendations.
  • Aggregated and sorted ratings to identify the top 50 books by popularity.
  • Serialized results with pickle for efficient downstream use and deployment.

Collaborative Filtering Based Recommendation

This module implements a user-based collaborative filtering recommendation system using cosine similarity on user-item rating matrices. I filtered the dataset to include only active users with over 200 ratings and books with sufficient user engagement, enhancing recommendation accuracy. The system transforms user ratings into a pivot table to calculate similarity scores between books, enabling personalized book suggestions based on user preferences. This project highlights my skills in matrix manipulation, similarity computations, and real-world model implementation using Python and scikit-learn.

  • Filtered users and books based on rating counts to reduce noise and improve model robustness.
  • Created a user-item matrix and handled missing data with appropriate imputation.
  • Computed cosine similarity to measure item-to-item closeness for recommendation.
  • Developed a recommendation function returning relevant book titles and metadata dynamically.
  • Saved model artifacts with pickle for efficient integration into web apps or APIs.
